Great photo organization

Photos are searchable, and browsable with folders and tags.

Full audit trail

Any change to the site can be undone, individually or in bulk.

Easy to install

Install in one step. See this 50-second install flyover (on a fresh Ubuntu server)

Easy to get photos in

Upload photos using an iOS or Android app, or direct from your SLR camera.

Responsive

Supports RSS, OpenGraph, OpenSearch and devices from Netscape 1.0 to iOS 8.

Integrates well with your site

Uses OpenID for login/registration and integrates easily with existing login and can serve images off of Amazon S3, Dropbox, remote HTTP, and other CDNs.

Camera Life is an open source software package you install and run on your own server running PHP and MySQL. This creates a beatiful experience for your visitors that will be viewing your website and seeing your collection. This gallery comes with a great theme out of the box but there are also great options for integrating the style of your own website and your existing user authentication if that is necessary.
